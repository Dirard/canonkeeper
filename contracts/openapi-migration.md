## Frontend Adaptation Notes

- Auth session reads move from /auth/me to /auth/session; response stays sanitized and no-store.
- Search moves from URL query to POST /projects/{projectId}/search body to keep private manuscript text out of URLs.
- Canonical chat streaming is now POST /chats/{chatId}/turns plus GET /chat-turns/{turnId}/events; old mixed stream operation is removed from the final operation set and documented only as a migration replacement.
- Shared Job is canonical for import, indexing/reindex, export, chat turns and agent runs; typed job endpoints are removed from the final operation set.
- Full manuscript export remains owner/admin only; editor/viewer negative cases must be preserved when UI is migrated.
