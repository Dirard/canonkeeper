## Deletion And Retention Coverage

- users/session records: server sessions invalidated on logout/rotation; audit retained redacted; cache-control no-store.
- projects/memberships/invitations: user-facing removal is immediately invisible; invitation tokens are hashed/redacted; access audit retained.
- manuscripts/books/chapters/annotations: immediate invisibility on delete; background purge completes within 30d unless legal hold is configured outside this contract; public restore API is out of scope.
- chats/messages/artifacts/suggestions/event logs: project-scoped; event logs expire after 24h; persisted chat resources purge within 30d of project/resource deletion; raw provider/tool traces are not canonical payloads.
- jobs/imports/search/index caches: project-scoped, invalidated on content/member/delete; working files purge within 24h after terminal state; metadata retained 30d.
- exports: download artifacts expire within 15m or first download and purge within 24h; export metadata retained 30d.
- audit records: redacted, tenant-scoped, retained 365d, never include raw manuscript/chat/provider payloads or secret values.
